# 角色聲線一致性與第二章第二柱澄清｜變更紀錄

**日期**:2026-07-21
**核准**:總監（實玩回饋：「第二柱看不懂」「旅人後段愈來愈文言」「章末寫鉤子很奇怪」）
**範圍**:第一章、第二章凍結劇本與其 runtime 文字
**性質**:文案與可讀性修正；不改物理、判定、證據配對、資源、分支或存檔

## 問題

1. 第二章 P2 使用「推力衰竭」與容器比喻，卻沒有先把理論押注的可測結果講清楚。
2. 第一章旅人在辯論中出現「便知、二石、擲下、敢問」等半文言，與前段現代白話聲線不一致。
3. 辛普里奧的老派書面語殘留「爾之」等理解門檻。
4. 第一、二章章末把編劇用語「鉤子」直接顯示給玩家，而且第二章的望遠鏡線索不等於第三章必然直接接續。

## 裁決與改法

### A. 第二章第二柱

對手命題改為：「向前的力沒用完以前，球只會往前；等它用完，球才開始往下掉。」

其可測預測為：若命題成立，水平拋出的球應比原地放下的球晚落。F3 改名為「一拋一放・近乎同時落地」，三輪量不到延後，因此反駁「先飛後墜」，而非籠統反駁「推力」。

### B. 聲線

- 旅人：一律現代白話；三處辯論長句重寫。
- 伽利略：兩處半文言保留為刻意模仿，但資料與劇本都明標「模仿」。
- 辛普里奧：保留「老夫」與莊重節奏，移除「爾之」等理解門檻。
- 建立 `02_設計/發現之前_角色聲線與對話規範_v0.1.md` 作為後續章節共同契約。

### C. 章末

- 第一章：改為「旅人筆記新增未解線索：它往前，又往下——兩件事，同時發生。」
- 第二章：改為「旅人筆記新增未解線索：兩片鏡片，能把遠處拉到眼前。」
- 「鉤子」只可留在內部製作文件，不得出現在玩家可見資料。

## 驗證門檻

1. JSON／JS 雙載體深等。
2. 第一、二章旅人台詞禁用表通過。
3. 伽利略半文言僅存在於明標模仿的指定節點。
4. 玩家可見資料不含「鉤子」。
5. 第二章 P2 仍由 F3 擊破，其他判定與狀態機不變。
