# 《發現之前》實驗證據所有權與認知進階原則 v0.1

**日期**：2026-07-22

**適用範圍**：全系列所有實驗、觀察、模型與公開論證章節
**目的**：確保玩家不是替科學家按完按鈕後自動領答案，而是親手把原始紀錄組成可辯護的證據。

---

## 一、不可退讓的核心

### 1. 原始結果不等於證據

跑完實驗只代表「有了紀錄」。只有玩家完成下列三件事，系統才可授予證據：

1. 選出要引用的紀錄；
2. 說明這些紀錄比較了什麼、控制了什麼；
3. 選出資料真正支持、且沒有超出邊界的主張。

因此禁止：跑滿次數就自動取得證據、NPC 代替玩家完成歸納、系統把正確結論寫在按鈕上。

### 2. 標準實驗循環

每個核心實驗依序經過：

> 問題 → 器材踏查 → 預測 → 取得原始紀錄 → 選資料 → 選概念／提出斷言 → 取得證據 → 帶進質詢

可依劇情省略非必要步驟，但不可省略「玩家提出斷言」。如果某章只剩操作與自動頒證，視為玩法回歸缺陷。

### 3. 器材踏查不是尋物遊戲

- 踏查的目的，是讓玩家知道每件器材控制哪個實驗條件；必要件不得藏在像素角落。
- 固定且唯一的骨架直接安裝並說明責任，不製造假選擇。
- 有替代件且會改變讀值時，才讓玩家選擇、比較與承擔代價。
- 踏查完成後，實驗室只呈現真正有意義的選項；已解釋的固定件標示「已固定｜無須更換」。

## 二、跨章認知進階

章節升級不是把數字和欄位越堆越多，而是提高玩家對「什麼算證據」的要求。

| 章節層級 | 玩家主要工作 | 完章應掌握的判讀 |
|---|---|---|
| 第一章：規律 | 從重複量測辨認數列並押下一筆 | 資料是否穩定、預測是否命中 |
| 第二章：比較 | 控制裝置與變因，選成對資料比較 | 只改一件事時，結果是否隨它改變 |
| 第三章：模型 | 從多種船況與觀察位置挑選來源，限定主張 | 同一事件可有不同描述；證據能排除什麼、不能證明什麼 |
| 後續章節 | 模型競爭、精度、異常、可辨識性 | 以最少額外假設解釋最多觀察，並保留未知 |

第三章因此不追求比前章更多的無意義數字，而要求玩家辨認：停船基準、穩速紀錄、加減速反例、船上／岸上兩份同步紀錄，各自能支持哪一層主張。

## 三、第三章落地規則

### G1｜航船桅頂落石

玩家須選至少三筆乾淨停船基準與三筆近似穩速落石，再選出「石頭放手後仍保留與船共有的前行」。加速窗、手放散布或不足筆數不得成立。

### G2｜封閉船艙比較

玩家須同時引用停船／穩速 × 滴水／拋接四格結果，再判斷兩種船況的局部現象近乎相同。只做一種現象或只看一種船況不得成立。

### G3｜加速／減速邊界

玩家須引用加速與減速兩筆相反偏移，判斷「共同運動一旦改變，相對落點就會偏移」。不得把偏移重新命名成石頭遺失的推力。

### G4｜雙參考物紀錄

玩家須引用同一事件的船上與岸上同步紙帶，完成時刻對齊與位移轉換，再主張「兩張軌跡可以同時正確，差別在參考物」。不得宣稱只有一張是真實。

## 四、視覺與回饋

1. **世界圖負責情境**：器材的位置、規模、使用方式與歷史氛圍由場景圖／物件圖呈現。
2. **程式圖負責物理**：讀值、落點、軌跡、同步標記與可切換參考物使用 HTML／SVG，避免把答案畫死。
3. **每種實驗有不同可見狀態**：停船、穩速、加速、減速、船艙滴水、拋接、船上紙帶、岸上紙帶不得只共用一個靜態圖示。
4. **紀錄與證據分色分階段**：原始紀錄是中性墨色；玩家斷言成立後才出現金色證據脈動與證據卡。
5. **證據可追溯**：旅人筆記的證據卡必須能展開來源紀錄、玩家選的主張與適用邊界。
6. **失敗要說人話**：回饋指出「少了哪組資料」「混入哪種船況」「這個主張超過資料能說的範圍」，不顯示內部代碼。

## 五、歷史誠實邊界

- 玩家參與史料空隙中的組裝、測量與判讀，不取代真實科學家。
- 教學代理、合理重建與虛構串接必須在章末史實頁揭露。
- 不得偽造精確歷史讀值並冒充原始史料；fixture 是可重現的遊戲模型，文件需明示其身分。
- 一個反對意見失效，不等於整套宇宙論已被證明。

## 六、出貨檢核

- [ ] 跑完操作但未斷言時，證據仍為未取得。
- [ ] 選錯來源、資料不足或概念過強時，狀態不部分成立。
- [ ] 正解位置不得形成固定規律。
- [ ] 舊存檔缺少新欄位時可安全補值。
- [ ] 器材踏查完成前不得直接進核心實驗；讀檔落在實驗內時可合理恢復。
- [ ] 桌機與手機橫屏都看得到來源資料、概念選項、斷言按鈕與錯誤回饋。
- [ ] 取得的證據能在旅人筆記回看來源與邊界。
- [ ] 自動測試含「完成全部實驗但證據仍未取得」的負向案例。
