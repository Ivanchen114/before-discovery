## 1. 章節視覺命題

**1590 年，比薩。證據在昏暗、老舊、充滿權威的世界裡逐步變得可見。**

- 伽利略是 26 歲、薪水微薄、聰明但正在卡關的青年教授，不使用 1635 年經典老年白鬍形象。
- 辛普里奧是 58 歲、博學而有尊嚴的資深教授，不是反派漫畫或降智丑角。
- 場景與器材必須有可操作性；斜面、水鐘、球、記錄紙不是背景裝飾。
- 視覺高潮不是魔法特效，而是人物終於被一組可重現的數據迫使停頓。
- 辛普里奧採老派書面語；他的時代感同時由姿態、服裝與禮節承擔，不靠難懂文言維持角色性。

## 2. 寫實歷史插畫方向

### 2.1 媒材與質感

- 寫實、解剖可信、帶手繪油畫筆觸的歷史插畫；不是照片，也不模仿特定在世藝術家。
- 材質必須可辨：磨損羊毛、亞麻白領、舊木、鉛／銅氧化、石灰牆、羊皮紙與蠟燭煙痕。
- 色彩以石灰灰、深胡桃木、炭黑學袍、舊紙奶油色為主；晨景加冷藍，實驗加琥珀燭光。
- 不用 steampunk、奇幻光芒、現代實驗室玻璃器材或巴洛克宮殿式浮誇裝潢。

### 2.2 構圖規則

- 場景圖一律保留 HTML UI 安全區；文字、數字與按鈕不得燒進圖。
- 人物視線與手勢應指向證據或器材，不讓玩家只看臉而忽略因果。
- 每張圖同時提供「無 UI 底圖」與整合截圖；runtime UI 由 Claude 實作。
- 辯論畫面不做勝利英雄姿勢；學生反應克制，重點是思考與動搖。

### 3.2 目前不可直接上線的理由

- 來源圖為 1672×941，低於正式場景母版 2560×1440 要求。
- 每張 PNG 約 2.0–2.2MB；它們是 proof 來源，不是 §2.3 的 runtime 衍生檔。
- 角色 A/B 已完成 proof，但正式母版尚未以 ≥2400 px 長邊重製並做髮絲人工清邊。
- 生成圖中的折扇、水鐘、支架與建築關係必須經人工結構／史料修整。
- 圖上偽文字只可視為筆觸；正式數據、手稿內容與 UI 一律由 HTML／SVG 顯示。

## 4. 完整第一章資產範圍

### 4.1 角色

| 角色 | 正式需求 |
|---|---|
| 伽利略（26） | 1 身份母版、3 姿勢、8 表情、袖口整理／持球／指數據／疲憊等手勢差分 |

| 辯論主持 | 1 半身立繪、2 表情 |
| 旅人 | 第一章預設第一人稱不露臉；手部或袖口只在必要互動出現，外觀差分另案 |

### 4.2 背景與狀態變體

1. 比薩大學迴廊：晨、午後。
2. 伽利略書房／工作室：日、夜、雨夜、深夜實驗四種光線狀態，共用幾何底圖。
3. 高塔頂層與塔下落點：黎明。
4. 舊城牆快試區：日間。
5. 阿諾河邊：黃昏。
6. 大學大講堂：一般、辯論中、支柱破裂後三種人群／光線狀態。
7. 比薩廣場塔影：尾聲黃昏。
8. 旅人筆記末頁／月球亞平寧：章末靜態插畫，不做預錄影片。

### 4.3 器材、證據與道具

- 大／小鉛球、大／小銅球、木球、木屑、同尺寸鉛／木球。
- 《論運動》手稿、德爾夫特來信、旅人筆記、證據卡 E1–E5／S1–S2。
- 可調傾角木槽：打磨／原木兩表面、三種傾角、球的起點／終點狀態。
- 水鐘：容器、細孔、集水杯與稱重／讀值表現；正式外形須先通過器材功能圖。
- 脈搏計時手勢、音格標記、量測尺、支架、楔塊與記錄紙。
- 水槽／水缸、繫球鏈條、辛普里奧折扇、辯論桌與證據展開紙。

### 4.4 程式原生視覺與必要動效

以下不生成人物影片，由 Claude 以 HTML／CSS／SVG／Canvas 實作，Sol 提供畫面規格並審核：

- 高塔兩球落下與近乎重疊的落地聲／時差。
- 綁縛悖論的兩種推論組裝。
- 斜面球運動、計時、四段讀值、比較與誤差特徵。
- 水中／空氣介質比較。
- 證據卡取得、旅人筆記、辯論支柱裂解與說服力量表。
- 章末石子拋體弧線與月球頁面淡入。

**影片政策**：第一章不需要全螢幕預錄影片。角色以表情／姿勢差分、分層鏡頭移動、光影與少量 FX 完成；宣傳片等完整版可玩後再做。

## 5. 從 Art Proof 到「可玩的完整版」

1. **總監 Style Lock**：確認或修改本批伽利略、辛普里奧、筆觸與色調。
2. **角色 A/B proof**：已完成；三表情 PNG 實測由 4,391,811 bytes 降至 2,077,796 bytes（少 52.7%），ART-ADR 草案建議混合制。
3. **史料／器材包**：逐場景確認 1590 比薩建築、服裝、斜面、水鐘、球與手稿來源。
4. **完整資產清單與 scene ID 對齊**：劇本、程式與美術使用同一 ID；先修已登錄的第一章劇本債務。
5. **分批量產**：角色 → 共用書房底圖 → 高塔／河邊／講堂 → 道具／證據 → 章末 CG。
6. **Claude 全章整合**：序幕、高塔、P1–P3、斜面、介質比較、最後反撲與尾聲全部可玩。
7. **視覺 QA／形成性試玩／Art Lock**：鍵盤、降低動態、文字放大、史實頁與學生閱讀卡點一併驗收。

Art Proof 可以現在平行進行；「完整版」則必須同時完成尚未實作的章節流程。美術不會把現有只有斜面＋P3 的灰盒偽裝成全章完成。
