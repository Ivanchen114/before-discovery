## 1. 章節視覺命題

**這一章不是把第一章的斜面換成另一台機器，而是讓一條看不見的路被玩家組出來、量出來、辯出來。**

- 木、沙、墨與紙是主要材質；視覺重心從「時間留下數字」轉為「運動在空間留下痕跡」。
- 器材圖讓玩家擁有裝置，但不替玩家診斷裝置。
- 辛普里奧已從拒絕數據的對手，變成會帶數據、會劃掉自己計算的同行；表情差分必須演出這個進化。
- 正確軌跡與判定仍由玩家完成；美術負責物件與歷史錯圖，不燒入答案。

## 2. 本批美術裁決

### ART-CH2-001｜伽利略 44 歲採新年代母版

不沿用 39 歲組。五年差距在一般背景 NPC 上可忽略，但伽利略是高曝光主角，且第二章的角色狀態從「急著證明」轉向「知道怎麼讓問題自己開口」。新母版只做微老化與神態沉穩，保留同一張臉、特徵痣、髮鬚與服裝語彙。

### ART-CH2-002｜彈射工坊採母版＋slot figure

- 中央持續顯示完整裝置母版。
- `release`、`edge`、`rangeBed` 以錨定於裝置周邊的透明零件縮圖顯示目前選件；這些圖不是像素對齊貼皮，不得硬疊到母版上。
- `launcher` 與 `heightRig` 是固定必要件，可由母版與空槽狀態表現。
- 空槽、替換與校準狀態由 HTML／SVG 外框、插槽與標記表現；不得在圖上畫 profile 名或「好／壞」。

### ART-CH2-003｜辛普里奧 `strikeout` 為整圖動作差分

正式版必須同時看見低頭、羽筆、紙與劃除動作。只有低頭表情的 v01 已廢止，不得掛載；runtime 使用 v02。

## 5. 程式整合契約

1. Runtime 根目錄為 `public/assets/`；完整 ID／path 見 `art/source/production/ch02/asset-manifest.json`。
2. 1608 場景禁掛 39 歲伽利略、58／72 歲辛普里奧；年代守衛應驗 scene → portrait ID。
3. 人物永不鏡像；站位錯誤改資料 mapping，不翻圖。
4. `strikeout` 使用 v02；v01 視為 deprecated asset。
5. 器材選件圖顯示於 slot figure／裝置快照；空槽不可偷偷顯示已裝零件。
6. S3 為史料例外；F2 數據、FR 疊線、落點、斜率與現代正確軌跡一律程式 SVG。

## 6. 尺寸與預算

- 背景：1920×1080 WebP，323KB。
- 證據卡：800×500 WebP，各 81–89KB。
- 對話人物：800×1200 透明 WebP，各 175–192KB；遠高於 170–220 CSS px 顯示需求。
- P0／P1 active runtime 共 20 件；連同一件已廢止的 v01，整個 `public/assets/ch02/` 約 2.7MB。
- 角色生成原生為 1024×1536；目前 runtime 足量。若未來做 4K 宣傳或印刷，不可把插值放大冒充原生高解析，須另做宣傳母版。
