## 問題分級

### A 級

無。沒有會阻止 P-5 選圖的身份錯誤、亂字、浮水印、現代物件或明顯肢體錯誤。

### B 級（Style Lock 後、量產前修）

1. **水鐘功能比例**：`ch01_prop_water_clock_detail_v01.png` 的集水桶大於天平秤盤，無法直接「把桶放上去秤」。v02 應改為可取下的小集水杯，或明確畫出「大桶倒入秤盤容器」的第二步；出水控制件也需避免現代水龍頭感。正式功能圖通過前，不鎖定這件器材外形。
2. **來源解析度**：場景 proof 多為 1672×941，低於正式母版 2560×1440；這是依任務書刻意延後的量產工作，不得把 proof 直接壓成 runtime 當完成品。
3. **透明髮絲**：A/B proof 的捲髮邊緣仍有少量綠幕溢色。正式 2400 px 角色母版需人工 1–2 px matte／清邊並在深、淺兩種背景檢查。
4. **群眾與建築史料**：辯論廳觀眾仍有生成式重複臉傾向；比薩迴廊與斜塔視線是額外探索圖，正式場景需另做建築來源包。

### C 級（可在量產模板處理）

- 頓悟表情的張嘴幅度是本角色的上限，正式版不得再向漫畫式誇張增加。
- UI 合成稿只驗證佈局；正式字級、鍵盤焦點、色彩對比與縮放仍由 HTML/CSS 實作驗收。
