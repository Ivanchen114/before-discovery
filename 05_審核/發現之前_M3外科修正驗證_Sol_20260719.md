## 2. A 級｜探索模式 E3.c 的斷言按鈕仍被 UI 隱藏

**位置**：`greybox/src/chapter-ui.js:115-123`，尤其第 122 行；實際 E3.c embed 為 `greybox/data/scenes.js:1180-1189`。

**重現步驟**：

1. 探索模式完成 E3.a 與換球 E3.b。
2. 劇情正確進入 `A2-3/e3c`，提示「用僅傾角不同的兩筆成立主張」。
3. 以銅小／打磨／中傾角／水鐘完成 run #3；此時狀態仍為 `E3:a●b●c○`。
4. 實驗台只顯示「斷言：與球重無關」；「斷言：隨傾角形式不變」不存在於可見 UI，繼續按鈕維持 disabled。

**最小修補**：斷言按鈕應由「目前 embed 要求」決定，不再由模式決定。例如在 `renderAll()` 的 incline embed 分支依 `v.until.e3` 更新：

- 其他階段：依需求顯示 B，不得讓探索模式的 C 必經段沒有 C。

修後須以探索模式在瀏覽器看見並點擊 C，確認 `E3:a●b●c●` 後才可算 A-1 關閉。不要只增加另一項直接呼叫引擎的 Node 測試。
