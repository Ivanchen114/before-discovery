## 0. 範圍

**含**:序幕→尾聲之全章可玩流程(劇本 v0.2 全部場景)、敘事引擎、信譽系統、高塔實驗、E4 對照實驗、三支柱+最後反撲之完整辯論、探索/學者模式分層、localStorage 存檔、章末史實頁。
**不含**:美術音效(灰盒紀律仍適用,R-ENV-04)、AI 功能、行動裝置、成就系統、多語、第二章。
**互動顆粒度原則**:斜面實驗沿用灰盒實驗台(R-EXP 全套);其餘實驗(高塔、E4)為**敘事內嵌之低顆粒互動**(選擇+固定結果),不做數值模擬——依憲章一,它們是證據取得與變因教學,不是核心實驗迴圈;核心迴圈唯一歸斜面。

## 1. 敘事引擎(R-NAR)

- **R-NAR-01 資料驅動**:全章敘事以場景圖定義於 `/data/scenes.js`(+`scenes.json` 鏡像,R-DATA-05 適用)。場景 ID 與節點標籤**逐字對齊劇本 v0.2**(如 `P0-2`、`A2-3.a`)。引擎通用,內容即資料。
- **R-NAR-02 節點類型**:`line`(台詞/演出)、`choice`(選項)、`system`(量表/證據事件顯示)、`embed`(內嵌子系統:實驗台/辯論廳/高塔/E4/修復事件)、`goto`。效果原語:`rep±n`、`gainEvidence(id)`、`setFlag(k,v)`、`gotoScene(id)`。
- **R-NAR-03 守衛與匯流**:選項與節點可帶 `require`(旗標/證據/信譽條件);不滿足之選項**隱藏**(非灰化,避免暗示)。分支必以顯式 `next` 匯流;全章分支結構須與劇本 v0.2 附錄 A 一致。
- **R-NAR-04 模式標記**:節點可標 `mode ∈ {all, explore, scholar}`;引擎依模式過濾。
- **R-NAR-05 結論詞彙 lint(刪台詞測試機器化)**:資料層維護**禁詞表**(初值:「平方」「1:3:5:7」「奇數」「與重量無關」「一樣快」)與**白名單節點清單**。自動測試檢查:禁詞出現於 NPC `line` 節點時,該節點必須在白名單內;白名單節點必須位於對應玩家操作節點(指認/斷言/選項)**之後**(依場景圖拓撲序)。新增台詞違反 lint 即測試失敗。白名單變更須記 changelog。
- **R-NAR-06 埋點事件**:`gainEvidence`、`rep 變動`、死路 A/B 抵達、辯論中止、章末,寫入 session 內事件log(記憶體+存檔),供形成性試玩回放觀察;不外傳(零資料蒐集不變)。

## 2. 信譽系統(R-REP)

- **R-REP-01**:信譽整數 0–5,起始 3,顯示於狀態列。變動事件以劇本 v0.2 附錄 A 為準(資料驅動,不寫死於程式)。
- **R-REP-02 歸零軟鎖**:信譽=0 → 標記 `repLocked`,關鍵 NPC 對話選項被鎖(顯示鎖定語)→ 強制進入修復事件【SC-R1】。
- **R-REP-03 修復事件**:SC-R1=完成任一次斜面實驗並「出示紀錄」→ 信譽回復至 1、解鎖,返回原節點。可重複觸發。
- **R-REP-04**:信譽在辯論中之扣減(A3-3.a、A3-5.trap)與說服力扣減**獨立記帳**,兩表各自顯示。

## 3. 高塔實驗(R-TWR)——場景 A1-3/A1-4

- **R-TWR-01 三選單**:丟什麼(`q1`)/在哪丟(`q2`)/找誰看(`q3`,設 `w ∈ {student, assistant, none}`)。
- **R-TWR-02 單變因勸阻守衛**:`q1=B(鉛+木)`→ 進勸阻節點(伽利略台詞)→ 返回 `q1` 且選項 B 標記「已勸阻」不可再選。筆記寫入「一次只動一個變因」。
- **R-TWR-03 執行與結果**:選項齊備 → 塔上演出(固定結果文本:兩聲近乎重疊、重者先一線)→ `gainEvidence(E1)`;E1 為**觀察卡**(含 w 註記),非數據 run,不進判定選集。
- **R-TWR-04**:w 影響 A1-5 之插入台詞與 P1 辯論之 E1 特殊回應措辭(僅文本差異,無數值差異)。

## 4. E4 對照實驗(R-E4)——場景 A2-4

- **R-E4-01 紙實驗(只動形狀)**:三個操作按鈕:攤平放手/揉團放手/上天平比對。三者皆執行(順序自由)→ 紙結論節點解鎖(玩家自選結論選項:「變的是形狀」/「變的是重量」——後者被天平結果駁回,重選)。
- **R-E4-02 球實驗(只動介質)**:兩個操作:空氣中放手(銅大+銅小)/水缸中放手。皆執行 → 球結論節點(玩家自選:「球變了」/「介質變了」)。
- **R-E4-03**:兩實驗皆完成且結論正確 → `gainEvidence(E4)`。E4 卡面記載兩組「單一變因差」摘要。
- **R-E4-04 誠實條款**:結果文本不得使用「完全同時」表述空氣中之銅球對照;一律「近乎同落/差一線」。真空表述必附條件(沿架構憲章四)。

## 5. 辯論系統全章擴充(R-DEB-09 起;R-DEB-01~08 原文適用於每根支柱)

- **R-DEB-09 支柱序列**:辯論資料新增 `pillars: [P1, P2, P3, FR]`,依序進行;每根支柱結構同 §6.1(三句證詞、弱點、追問文),weakTo:**P1=(E4→證詞1-2)、P2=(E2→證詞2-2)、P3=(E3.b→證詞s2,沿切片)**。前一支柱破裂才進下一支柱。
- **R-DEB-10 特殊回應表**:資料層支援 `special` 條目——三元組→(回應文,量表效果,旗標)。初值一條:P1 之 (E1,-,證詞1-2) → 「又是幾乎」回應,**量表不扣、支柱不動**,筆記提示解鎖。此表資料驅動,可擴充。
- **R-DEB-11 追問內嵌選項**:證詞之 `press` 得攜帶 `choice`(如 A3-3 追問 2-3 之「憑誰立論」):選項效果沿 R-NAR-02 原語(A 選項:rep−1、persuasion−1、返回選項)。
- **R-DEB-12 最後反撲(FR/E5)**:P1–P3 全破後觸發。
  - 〔探索〕兩步引導選項(`A3-5.e1/e2`),答錯僅重選,不扣量表;
  - 〔學者〕三槽組鏈:槽位=①各傾角皆成立②增陡不改形式③垂直為極限方向;干擾項三條(空話/次數/訴諸權威),**選入干擾項該槽退回且說服力 −1**;
  - 組鏈完成 → **trap 抉擇節點**(`A3-5.trap/honest`):「量過」→ 說服力 −2、信譽 −1、強制轉 honest;「沒有」→ 直接 honest;
  - honest 完成 → `gainEvidence(E5)`、FR 標記 resolved。
- **R-DEB-13 勝利與中止**:勝利=P1∧P2∧P3 broken ∧ FR resolved → 判定演出(A3-6)→ 尾聲。說服力歸零(任一階段)→ 中止(R-DEB-07 語意)→ 返回第二幕,**全部支柱與證詞狀態保留**;再入辯論從第一根未破支柱續,量表重置 5,已破支柱以一行帶過(不重打)。
- **R-DEB-14**:E3 出示時之子項展開、E3.b 守衛,沿切片 R-DEB-03/04/05 不變。

## 6. E3 敘事掛鉤(R-HOOK)——場景 A2-3

- **R-HOOK-01**:`E3.a 首次點亮`事件 → 觸發場景 A2-3(伽利略模仿橋段)。場景內選項 `b(吵渴望)`被駁回重選;選項 `a(換球)`→ `embed` 實驗台(預選銅小,玩家仍可自由改配置——**不鎖自由度**,僅預選)。
- **R-HOOK-02**:`E3.b 點亮`→ A2-3 收尾台詞與筆記更新;〔學者〕加掛 E3.c 支線(同結構,預選變傾角)。E3 確立(a∧b)後才開放 A2-5(辛普里奧來信)。

## 7. 存檔(R-SAV)

- **R-SAV-01**:localStorage 單一鍵 `bd_ch1_save`,內容={schemaVersion, mode, 敘事游標(場景+節點), 三層狀態(R-STA-01~03 全量), rep, flags(w 等), 事件log, 章末兩題文本}。每場景轉換與每次 embed 子系統返回時自動存。
- **R-SAV-02 容錯**:parse 失敗或 schemaVersion 不符 → 提示後開新局,壞檔改鍵名備份(`bd_ch1_save_corrupt`),不靜默覆蓋。
- **R-SAV-03**:標題畫面提供「繼續/新遊戲(需二次確認)」;新遊戲清空存檔。
- **R-SAV-04**:存檔往返測試:任意狀態 serialize→deserialize→深度相等,入自動測試。

## 8. 模式分層(R-MODE)

- **R-MODE-01**:新遊戲時選擇 探索/學者;存入存檔;v0 中途不可換。
- **R-MODE-02 差異表(規範)**:

| 環節 | 探索 | 學者 |
|---|---|---|
| 斜面配置 | 全自由(同灰盒) | 全自由+音格計時解鎖 |
| 死路 B 診斷 | 可複選診斷選項(A2-2) | 相同(診斷即教學,不因模式剝奪) |
| E3 | a+b 必經 | a+b 必經+c 支線 |
| 最後反撲 | 兩步引導 | 三槽組鏈+干擾項 |
| 綁縛悖論加問 | 無 | A1-7 加問(「矛盾何時消失」) |

  註:音格於探索模式**不出現於選單**(非灰化);切片通過矩陣不受影響(矩陣以配置空間全集定義,測試不分模式)。
- **R-MODE-03**:模式差異一律資料標記(R-NAR-04/資料欄位),禁止程式內 if 散落(單一過濾機制)。

## 9. 章末與史實頁(R-END)

- **R-END-01**:A3-6 判定 → 尾聲(E-1、E-2)→ 兩題自由輸入(存入存檔,不評分,R-UI-02 語意)→ 史實與虛構頁。
- **R-END-02**:史實頁資料驅動(`/data/histfacts.js`+json 鏡像),條目=劇本 v0.2 史實頁表全文;每條含狀態標籤(史實/傳說/戲劇改編/教學代理)。
- **R-END-03**:章末顯示總天數與(若曾)辯論中止次數——與 R-UI-02 併版面。
