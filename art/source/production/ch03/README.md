# 第三章正式美術交付

章名：第三章《船艙裡的靜止》
美術基線：寫實歷史插畫感、1640 年馬賽、低飽和暖褐與海灰藍、無現代物件、無畫面內文字。

## Runtime 資產

- 8 張 1920×1080 WebP：馬賽港清晨、繫泊桅杆甲板、穩速航行甲板、密閉船艙、變速甲板、雙紙帶比較桌、碼頭公開演示、1642 印刷室。
- 7 張 1920×1080 實驗互動底板：G1 停船／穩速桅頂、G2 船艙、G3 加速／減速甲板、G4 雙紙帶、G5 公開演示。它們只負責歷史空間與器材；水滴、石球、紙帶軌跡與證據狀態由程式依玩家操作疊加。
- 3 張 900×1200 透明 WebP：48 歲伽桑狄、約 50 歲船長、17 歲艾蒂安。
- 原始母版保留於 `backgrounds/` 與 `characters/`；runtime 位於 `public/assets/ch03/`。

## 生圖構圖約束

- 所有場景圖均保留底部約 30% 對話安全區；不把關鍵器材或人物放在最底部。
- 船桅落石使用手掌大小石球，不畫砲彈或巨球；公開演示的重點是桅杆、桅腳沙盤與預測木籌。
- 雙紙帶場景只畫可供 HTML/SVG 疊加的紙帶與鼓點標記，不把物理結論畫死。
- G1–G4 禁止把落點或軌跡畫死在底板；同一底板必須可支援錯誤預測、失敗紀錄與正確結果。G5 是證據邊界，不冒充新的物理實驗。
- 角色皆為獨立透明半身／全身像，程式永不鏡像；衣著依 1640 年馬賽與教士／水手／助手身份區分。

## 轉出工具

- `export_backgrounds.py`：母版置中裁成 16:9，輸出 1920×1080 WebP。
- `clean_checkerboard_alpha.py`：移除生成圖棋盤底，輸出透明 PNG 與 900×1200 WebP。
- `export_dialogue_busts.py`：從透明全身母版裁出 3:4 對話半身像；全身母版保留，不拿 CSS 放大硬撐。

資產清單與場景映射以 `greybox/data/assets.json` 為唯一 runtime 來源；程式測試鎖住 17 場與三角色，不另維護第二套表。
