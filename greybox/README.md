# 《發現之前》灰盒切片(bake-off 範圍)

斜面實驗+一場論辯(支柱 P3)。法源:灰盒功能規格書 v0.1.1、架構計劃書 v0.3、協作流程 v0.1.1。

## 執行

雙擊 `index.html` 即可遊玩(R-ENV-01,無建置、無網路請求;file:// 直接開)。

## 測試

`npm test`(零外部依賴,需 Node 18+)——含 54 配置矩陣一致性、安全邊際、狀態轉移、鏡像一致性,共 16 項。
或開啟 `tests/tests.html`(除鏡像一致性外之全部測試)。

## 玩法(灰盒,無教學演出)

1. **實驗台**:配置球/槽面/傾角/計時 → 執行。每次執行消耗天數(脈搏 1/水鐘 2/音格 3)。
2. **旅人筆記**:勾選同配置的 1–3 筆 run → 輸入「第五段增量預測值」→ 認證規律(成功=主張)。用兩筆僅「球」相異的成立主張執行斷言 → E3.b。
3. **辯論廳**:追問找線索,出示 (證據, 子項, 目標證詞)。正解需 E3.b。錯誤出示扣說服力;歸零中止,可補證後再入。

## 結構

- `/src` 引擎(純函式狀態機)與 UI;`/data` fixture(.js 執行載體+.json 規範鏡像);`/tests` 自動化測試;`/docs` 債務清單與 wireflow;`decisions.md` ADR。
