# 瀏覽器人工驗收清單(R-ENV-01/R-UI-01/02 + 程式審查 B-1/B-2)

**背景**:自動化測試(node)覆蓋引擎與資料層;以下項目依零依賴紀律不引入 headless 瀏覽器,以人工簽核。每次簽核記錄:日期/瀏覽器與版本/簽核人/結果。

## A. 雙擊可玩(R-ENV-01)

- [ ] Chromium 系(Chrome/Edge):檔案總管雙擊 `index.html` 開啟,狀態列顯示「天數:0」,三面板可切換,執行實驗有讀值。
- [ ] Firefox:同上。
- [ ] 開啟 DevTools Console:無紅字錯誤(file:// 下不得出現 CORS/fetch 錯誤)。

## B. 鍵盤完整通關(B-1)

- [ ] 只用 Tab/Shift+Tab/空白鍵/Enter/方向鍵完成:配置實驗→執行→切到筆記→勾選 run→輸入預測→認證→勾兩筆主張→斷言 b→切到辯論廳→出示 (E3,b,s2)→勝利畫面兩題可輸入。
- [ ] 全程可見焦點框(未被 CSS 移除)。

## C. 讀屏抽查(B-1,任一:NVDA/VoiceOver)

- [ ] run/claim 勾選框朗讀含編號與配置(非「未命名核取方塊」)。
- [ ] 認證成功/失敗訊息自動被朗讀(role=status);辯論紀錄新行被朗讀(role=log)。
- [ ] 頁籤朗讀含選取狀態(aria-selected)。

## D. B-2 回歸(視圖切換不清空選取)

- [ ] 執行 ≥2 筆 run→勾選 2 筆→切換「累計視圖」再切回「增量視圖」→勾選狀態保留→直接按「比較」成功。
- [ ] 認證產生主張後(表格重繪),原先勾選的 run 仍保持勾選。

## E. 三面板同步與收尾(R-UI-01/02)

- [ ] 在辯論廳扣量表後切回實驗台再切回,量表數值一致;狀態列與面板即時同步。
- [ ] 勝利畫面顯示總耗天數,兩題自由輸入可打字,重新整理後歸零(R-STA-05 記憶體存放,屬預期)。

## F. 完整第一章(M3,chapter.html)

- [ ] 雙擊 `chapter.html`:標題畫面→新遊戲(兩種模式各跑一次)。
- [ ] 純鍵盤通關全章:序幕選項→高塔三選單→綁縛推導→幕間→實驗台(認證+斷言)→E4 兩實驗→辯論(追問/出示/反問選項/FR/trap)→判定→兩題輸入→史實頁→終。
- [ ] 辯論中止路徑:故意錯誤出示×5→中止→離場→補實驗→重返(量表重置、已破支柱保留)。
- [ ] 存檔斷點:於第二幕中途關閉分頁→重開→「繼續上次進度」→紀錄與狀態完整。
- [ ] 說謊陷阱:trap 選「量過。」→ 雙重代價正確顯示;信譽歸零時被帶去修復事件並返回。
- [ ] 史實頁表格可讀;章末總天數顯示。

## F2. 書信碼(R-SAV-05/CR-001,2026-07-19 增;兩殼各驗一次)

- [ ] 有進度時標題畫面出現「匯出書信碼」;按下→欄位填入+已複製(或提示手動複製)+下載 .txt。
- [ ] 把碼貼到另一瀏覽器/另一台機器的「貼碼續玩」→ 進度、場景、實驗紀錄完整接續。
- [ ] 貼壞碼(亂改幾個字)→ 顯示「格式損壞/版本不符」,**本機既有進度不受影響**,「繼續」仍可用。

## G. 舞台版(stage.html,2026-07-19 增)

- [ ] 雙擊 `stage.html`:標題→新遊戲→**開場第一句自動開演**(不應出現空白對話框)。
- [ ] 對話逐字跳出;打字中點畫面=跳完整句;再點=下一句;Enter/Space 同義且不會在打字中誤觸底層按鈕。
- [ ] 旁白(斜體)與系統行=整句淡入,不逐字。
- [ ] 系統設「減少動態」後重開:全部即顯、無淡入、無閃爍游標。
- [ ] 場景切換:有圖場景(P0-1 騎樓/INT-1・A2 工坊/A3 講堂)背景淡入;無圖場景=素底+場景名,不破版。
- [ ] 立繪:辛普里奧/伽利略發言時出現;玩家發言時壓暗;無圖角色發言時退場。
- [ ] 實驗台/回顧/史實頁=羊皮紙面板;選項=右下疊卡(≥44px 高)。
- [ ] 存檔互通:stage.html 存檔→chapter.html「繼續」可接續,反之亦然。
- [ ] 視窗極端比例(超寬/直式)letterbox 正確,對話框不出界。

## G2. 第一輪視覺修正(2026-07-19 總監裁決;桌機 1280×720+手機橫屏約 844×390 各驗一輪)

- [ ] 對話字級:桌機肉眼舒適(約 30px 級);手機橫屏 20–24px;200% 系統文字放大下不破版。
- [ ] 長台詞自動分頁(▼ 翻頁),對話框內**永不出現捲軸**;每頁不超過約三行。
- [ ] 標點節奏:逗號微停、句號/問號明顯停;點擊仍可一鍵跳完;reduced-motion 全顯。
- [ ] 句子顯示完畢後,按 Enter/Space 可直接推進;表單輸入、選項、筆記開啟時不誤觸。
- [ ] 半身像:掛對話框左緣、向上突出(桌機約 170–220px 高);柔邊墨暈,**無方形米色邊界**;玩家發言時壓暗;無圖角色退場;無任何鏡像翻面。
- [ ] 旅人筆記=全畫面筆記本:桌機左頁證據+實驗快照/右頁完整紀錄;手機橫屏「證據/紀錄」分頁;按鈕與分頁 ≥44×44px;Esc 或關閉鈕退出,焦點回「旅人筆記」按鈕;開啟時舞台暫停壓暗、打字暫停,關閉續打。
- [ ] 手機橫屏 HUD 收斂(信譽/天數/說服力+按鈕),頂端不塞滿。
- [ ] 網路面板確認:非當前/下一場景的大圖**未被預載**(首屏 3MB 預算)。
