旅人筆記・辯論備忘

  1. 戰場——對面那位用三根支柱撐起他的理論。支柱不倒,理論不倒。
  2. 先偵察,再開火——「問到底」找出證詞真正依靠的前提;讀洞見與手牌上的白話摘要,再選一張能檢驗同一件事的證據。
  3. 以證據回擊——先點一句證詞,再從手牌選一張證據;行動鈕會把你的反駁完整說一遍。打中弱點=支柱裂;打錯=說服力 −1(第一支柱容你一次免扣試射)。
  4. 說服力歸零=今日辯論中止——不是缺證據。你會與伽利略複盤剛才的錯配，再入時已破的支柱不必重打。
  5. 鐵則——別說謊。被抓到的那一句,比整場辯論都貴。

旅人筆記・實驗備忘

  1. 器材——銅球沿斜槽滾下;水鐘在旁滴水計時。滴下等量的水=等長的時間。
  2. 量什麼——把滾動切成四個等時段,記下每段滾過幾格刻度。紀錄簿裡的「第 1–4 段」就是它們。
  3. 怎麼做——選配置 → 執行實驗 → 勾 1–3 筆同配置紀錄 → 押「第五段」會滾幾格 → 認證。押中了,規律才算你的。
  4. 計時器有脾氣——脈搏便宜但會抖;水鐘較穩,過快時可能固定偏向一側;音格最花時間也最精確。
  5. 認證門檻——前四段形狀與第五段預測兩項偏差都必須 ≤12%;失敗紀錄仍會留下,讓你診斷是哪裡出了問題。
  6. 備忘——每次實驗都花天數;筆記簿以墨水書寫,寫下的紀錄不可刪。
