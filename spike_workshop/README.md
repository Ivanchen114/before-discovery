## 系統摘要

4 typed slots(槽體/釋放/計時/標尺);釋放=門閂|手放、計時=水鐘|脈搏;校準二項(零點/標尺)正偏二態 → **16 組裝狀態,確定性映射 4 個 `apparatusProfile`**:

| profile | 支配條件 | 視覺語言(≥2 線索) |
|---|---|---|
| 尾段模糊/混合 | 標尺未校 | 霧帶+空心雙候選+「?」 |
| 系統偏移 | 零點未校(標尺已校) | 緊密同形+平行位移箭頭 |
| 隨機抖動 | 雙校但手放或脈搏 | 散點+短鬚線 |
| 乾淨 | 門閂+水鐘+雙校 | 貼參考虛線 |

三種受控自由:替代件選擇、裝配順序、校準時機。提示載體序:數據異常→筆記中性觀察(探索自動/學者索取)→(切片不含伽利略問句層,正式章才接)。

4. 重玩主動換路徑(場測)
5. 純組裝耗時紀錄(場測;正式章驗 <20%)
6. 16 態確定性重播+4 profile 視覺不矛盾(**已由 tests.mjs 自動把關**)
